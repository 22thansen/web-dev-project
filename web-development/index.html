Tanner's Web Projects

Hey my name is Tanner Hansen, and I like coding and have done a little bit of game development, but it's mostly just screwing around. I used to use GameMaker Studio 2 to make stuff but I've fallen out of practice. I did try Monogame, but I think it is too low-level and is good for games where you need more control and no black-boxes. I would try Unity, but all my devices are criminally low-end, and I think it would take too long to get anything up and running. The middle ground I've found is Godot, and I am beginning to learn how to use it. I've dipped my toes in it a little and I think this will be the engine for me: It is fast, leightweight, easy to develop (from what I've heard), and is completely free and open-source!

My plans for a game I was thinking about called The Legend of Mace (LoM), is now to make it in Godot, with a flat color 3D style (I may even add a pixel shader over it if I find it looks good). If I stuck with 2D on GMS2 I would probably keep running into unnecessary headaches that distract from the goal of gamedev -- making a game!

So that means I will have to learn 3D modelling, likely via blender. I've never made any sort of 3D game before; my only experience is when I tinkered in Roblox Studio, so this will be a new adventure. I might document my adventure, but I'm probably too lazy haha. Links to my itch.io page below.

As for this assignment it's just some projects in web-dev i've been doing. There's nothing too special about them.
