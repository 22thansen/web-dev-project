Colors & Fonts

Definition List:

CSS
The thing that makes websites look good and unique. It styles webpages.
HTML
The content of the webpage. It doesn't look very good without CSS
HTTP
some magic text you put before a url webpage. i think it helps with giving the content to your browser, cause http stands for Hypertext Transfer Protocol
HTTPS
Same thing as HTTP, but that "S" means it's secure.
Internet
Magical place of learning, levity, business, and lots of cats. It connects everybody from around the world.
IP Address
Every computer has one, it's like the ID of your computer. It's needed for internet stuff.
URL
URL means Uniform Resource Locator. It's sort of like the link you type into the address bar in your web browser
W3C
Means World-Wide-Web Consortium. I think it's the people who are in charge of internet standards.
Web-Safe Colors
Only a handful of colors, but they are the safest colors to use since every computer should be able to use them to browse the web.
WWW
The World Wide Web. it's usually found in a URL after the "http(s)://".
