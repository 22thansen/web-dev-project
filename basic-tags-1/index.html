Basic HTML tags

HTML (Hyper Text Markup Language) is a tag-based language. A tag is a word inside of angle brackets. Most HTML tags have closing tags. A closing tag looks the same as an opening tag with a slash inside the opening bracket.

A webpage can be created by typing HTML tags into Notepad and then saving it as an .html file. To do this, name the file something without spaces and then add the .html extension to the end of the file name.

HTML Skeleton Tags

  1. html - Stars and ends the document.
  2. head - Contains information about the document.
  3. title - Places a page title in the page tab of your web browser.
  4. body - Contains all the visible content of the webpage.

Basic HTML Tags

  • h1 - Creates largest size heading.
  • p - Creates a paragraph.
  • br - Places a line break into the page.
  • hr - Creates a horizontal line on the page.
  • ol & ul - Creates numbered and bulleted lists.
  • li - Adds additional list items to a list.

Attributes

HTML tags can have attributes, which provide additional information for that tag. An attribute is added inside the opening tag, after the tag name and before the closing angle bracket. In earlier versions of HTML, there are separate attributes for some effects, like the following:
p style="background-color: blue; color: white;"
